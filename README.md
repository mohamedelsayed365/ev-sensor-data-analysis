# Simulated Electric Vehicle Sensor Data Analysis

## Overview
This project simulates sensor data from an electric vehicle and performs exploratory data analysis using Python.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## Features
- Synthetic data generation
- Statistical analysis
- Data visualization
- Correlation analysis

## Key Insights
- Distribution of vehicle speed
- Relationship between speed and battery temperature
- Statistical summary of motor torque
